TictacToe
TTT is a two player game played on a 3 x 3 board. Each player take a turn a marking a square on the board. First player to get 3 marks in a row including diagonal wins. If all 9 available squares are marked and no player has 3 marks in a row it is a tie.

1. Display the initial empty 3x3 board
2. Ask the user to mark a square
3. Computer marks a square
4. Display updated board
5. If winner, display winner
6. If board is full, display tie
7. If neither winner nor board is full loop back to #2
8. Play again?
9. Go to #1 if yes
10. Good bye
